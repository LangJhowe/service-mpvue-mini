## 顺风车-小程序（service-mpvue-mini）

service-mini是基于`mpvue`构建的顺风车平台，已经上线（小程序码见下边），欢迎Star。

## 技术栈（2.0）

前端：小程序、uni-app、Vuex、flyio、font-awesome


## 技术栈（1.0）

前端：小程序、mpvue、Vuex、flyio、font-awesome

后端：Java、SpringBoot、MySQL


## 构建步骤（1.0）

``` bash
# 安装依赖
npm install

# 开发模式启动
npm run dev

# 打包
npm run build

具体步骤参见 mpvue 官网
```
